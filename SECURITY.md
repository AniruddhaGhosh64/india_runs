# Security Policy

## Supported Versions

Currently, only the main branch (latest release) is actively supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take the security of our project seriously. If you discover a security vulnerability, please do NOT report it by creating a public GitHub issue.

Instead, please send an email to the project maintainers (or reach out via the provided contact information in the submission details). 

We will acknowledge receipt of your vulnerability report and strive to send you regular updates about our progress. If the issue is verified, we will work to release a patch as quickly as possible.
